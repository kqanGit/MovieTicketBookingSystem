Detailed Description

Concrete implementation of movie entity in the booking system.

Represents a movie with all its metadata including basic information, ratings, and associated show times. This class implements the IMovie interface and serves as the primary movie data model.

The Movie class encapsulates:

  • Basic movie information (title, genre, description)
  • Rating system integration
  • Show time management
  • Unique identification
Design Patterns Used
  • Entity Pattern: Represents a domain object with identity
  • Data Transfer Object: Transfers movie data between layers
Invariant
id >= 0 (0 for new movies, positive for persisted movies)
rating >= 0.0 && rating <= 10.0
!title.empty()
