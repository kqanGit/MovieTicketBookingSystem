Detailed Description

Represents a movie ticket booking record.

This structure encapsulates all essential information about a booking made by a user, including movie details, showtime, and booking identification.

The Booking struct follows the Data Transfer Object (DTO) pattern, providing a simple container for booking data that can be easily passed between layers of the application.

Key responsibilities:

  • Store booking identification and metadata
  • Link bookings to specific movies and showtimes
  • Provide immutable booking record structure
  • Support booking history and reporting features
Note
This is a value type designed for efficient copying and storage
