Detailed Description

Concrete implementation of user authentication service.

This class provides the core authentication functionality for the movie ticket booking system. It implements the ILoginService interface and handles user login validation through integration with authentication repositories.

Key Features:

  • Username/password authentication
  • Repository pattern integration
  • Optional return type for safe failure handling
  • Support for guest and authenticated modes
  • Account information retrieval
Design Patterns Used
  • Strategy Pattern: Implements ILoginService interface
  • Repository Pattern: Uses IAuthenticationRepository for data access
  • Dependency Injection: Repository injected through constructor
Usage Example
IAuthenticationRepository* authRepo = new AuthenticationRepositorySQL();
LoginService loginService(authRepo);
auto result = loginService.authenticate("john_doe", "password123");
if (result.has_value()) {
AccountInformation account = result.value();
std::cout << "Login successful for: " << account.getUsername() << std::endl;
}

Authenticate user with username and password.

Validates user credentials against the authentication repository and returns account information if successful.

Parameters
usernameUser's login username
passwordUser's login password (plain text)
Returns
std::optional<AccountInformation> Account info if successful, empty if failed
Return values
std::optionalwith AccountInformation if authentication succeeds
std::nulloptif credentials are invalid or repository unavailable
Precondition
repo != nullptr for successful authentication
username and password are non-empty strings
Postcondition
No side effects on authentication failure
Account information contains valid user data on success
Security Considerations
  • Passwords should be hashed before storage (repository responsibility)
  • Failed attempts should be logged for security monitoring
  • Consider rate limiting for brute force protection
Warning
Passwords are currently handled as plain text in this interface
